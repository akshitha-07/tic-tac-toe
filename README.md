# Tic Tac Toe AI

An AI-powered Tic Tac Toe game built using Python and the Minimax algorithm.
The AI uses adversarial search to always make the optimal move, creating an unbeatable gameplay experience.

## Features

* Intelligent AI opponent using the Minimax algorithm
* Optimal move selection through recursive game tree evaluation
* Terminal state and winner detection
* Utility-based board evaluation system
* Interactive graphical interface using Pygame

## Tech Stack

* Python
* Pygame
* Minimax Algorithm
* Game Theory Concepts

## How It Works

The project uses the Minimax algorithm to simulate all possible future game states and select the best possible move for the AI player. The AI evaluates board states recursively and chooses actions that maximize its chances of winning while minimizing the opponent's advantage.

## How to Run

```bash
python runner.py
```
